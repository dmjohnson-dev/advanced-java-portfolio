# D387 – Advanced Java | Portfolio Project

This project is a Java-based backend application demonstrating advanced object-oriented programming concepts, application structure, and real-world Java development practices. It is published as a **portfolio version** of an academic project completed as part of the Computer Science program at Western Governors University.

---

## What This Project Demonstrates

- Advanced Java programming techniques
- Object-oriented design (encapsulation, inheritance, polymorphism)
- Layered application structure
- Data handling and validation
- Exception handling and defensive programming
- Clean code organization and readability
- Version control workflows including branching and merging

---

## Technologies Used

- **Java**
- **Java Collections Framework**
- **Object-Oriented Programming (OOP)**
- **Maven**
- **Git / GitHub**

---

## Project Overview

The application focuses on implementing business logic using advanced Java constructs while maintaining a clean and maintainable architecture. Emphasis is placed on correctness, scalability of logic, and adherence to best practices expected in professional Java development environments.

Key features include:
- Structured domain models
- Clear separation of concerns
- Robust input handling
- Meaningful exception and error management
